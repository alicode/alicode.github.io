Flask Config 載入方式

Flask config 內建變數很多,可以參考 官方Builtin Configuration Values,中文翻譯可以參考內置的配置值,這裡用DEBUG這個變數說明Config載入方。

以簡單程式碼為範例說明

預設沒有指定DEBUG是表False,不開啟程式debug模式。

執行之後可以看到 ,預設沒有debug模式。

若有開啟debug模式,如下:

  • 以陣列方式(寫在code裡)
  • 加入參數到app.run() (寫在code裡)
  • 以傳入模組物件方式 (寫在code裡)
  • 以傳入模組物件方式 (寫在另一個檔案裡;config.py)
    config.py 或是也可這樣表示
  • 透過檔案名稱方式載入
  • 透過系統的環境變數載入
